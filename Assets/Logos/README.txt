EMCI LOGO PACKAGE — 600 DPI EXPORTS
==================================

Official organisation name used:
Enhanced My Career Insights (Pilot Program)

Master files:
- 01_Vector_Master_SVG contains scalable, outlined vector artwork.
- SVG masters are the source of truth for resizing and professional output.
- Typography has been converted to vector outlines; no external font files are required.

Raster exports:
- 02_PNG_Transparent_600DPI: transparent high-resolution assets.
- 03_PNG_White_Background_600DPI: presentation/document-ready assets.
- 04_TIFF_Print_CMYK_600DPI: print-oriented CMYK TIFF files, tagged at 600 DPI.
- 05_Digital_Icons: portal/favicon-sized emblem exports.
- 06_Preview: presentation sheet and original generated concept reference.

Variants included:
- Primary official-name colour lockup
- Primary official-name black/single-colour lockup
- App-header colour lockup with “Student Management Platform”
- Acronym-only colour lockup
- Standalone emblem colour and black variants
- Reverse-white lockup for dark backgrounds

Colour palette:
- Deep Navy: #102B4C
- Slate Blue: #3C6178
- Restrained Teal: #0D817D
- Single-colour Black: #111827

Important:
The original generated logo board was a raster concept. This package recreates the concept as clean vector artwork for usable exports, and uses the official EMCI expansion supplied in the project brief.
